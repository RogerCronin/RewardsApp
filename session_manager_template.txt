- timeWhenDelete could be like a Unix timestamp or something similar

boolean confirmSession(sessionID, accountID)
- returns true if sessionID exists and is mapped to the correct accountID

String createSession(accountID)
- returns a new session ID that is tied to a specific account ID

void garbageCollect()
- automatically deletes session IDs that are over whatever_constant_here mins old
